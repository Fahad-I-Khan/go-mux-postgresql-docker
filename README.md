## To use this API 

### In terminal run this docker cmd's.

```
docker compose up -d go_db
```

```
docker compose build 
```

```
docker-compose up go-app 
```

### In another terminal go into the folder.

### Run this python cmd.

```
python3 -m http.server 8001
```

### In browser 

```
http://localhost:8001/
```
